LiteFlow从2.8.5版本开始，提供了条件组件的定义。

条件组件，也可以称之为IF组件，返回是一个true/false。可用于`IF...ELIF...ELSE`等关键字。

关于`IF...ELIF...ELSE`表达式的用法，可以参考[条件编排](/pages/v2.9.X/e76999/)这一章。

比如一个IF三元表达式，如下所示，x就是IF组件，为真，执行a，为假，执行b：

```xml
<chain name="chain1">
    IF(x, a, b);
</chain>
```

x节点的定义，需要继承`NodeIfComponent`：

```java
@Component("x")
public class XCmp extends NodeIfComponent {
	@Override
	public boolean processIf() throws Exception {
	    //do your biz
		return true;
	}
}
```

条件节点的内部可以覆盖的方法和this关键字可调用的方法见[普通组件](/pages/v2.9.X/8486fb/)这一章。
