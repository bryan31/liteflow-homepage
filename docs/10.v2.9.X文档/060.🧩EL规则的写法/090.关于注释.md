在LiteFlow的EL规则写法里，你也可以写注释。注释的写法有2种

单行注释

```xml

<chain name="chain1">
    //我是注释
    THEN(a, b, WHEN(c, d))
</chain>
```

多行注释

```xml
<chain name="chain1">
    THEN(
        //我是注释
        a, 
        b, 
        /**
        * 我是多行注释
        * 我是多行注释
        **/
        WHEN(c, d)
    )
</chain>
```
