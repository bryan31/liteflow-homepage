LiteFlow要求的Springboot的最低的版本是2.0。

支持的范围是Springboot 2.X ~ Springboot 4.X。

针对不同的Springboot大版本，LiteFlow提供了两个不同的starter，请根据你的Springboot版本选择对应的依赖：

| Springboot版本           | LiteFlow提供的starter依赖           | JDK要求      |
| ------------------------ | ----------------------------------- | ------------ |
| Springboot 2.X ~ 3.X     | `liteflow-spring-boot-starter`      | JDK8及以上（Springboot 3.X需JDK17及以上） |
| Springboot 4.X           | `liteflow-spring-boot4-starter`     | JDK17及以上  |

当然如果你使用了Springboot 3.X，相应的JDK版本也要切换为JDK17及以上版本。

而Springboot 4.X本身要求的最低JDK版本为JDK17，所以使用`liteflow-spring-boot4-starter`时，JDK版本也必须为JDK17及以上。

:::tip 注意
Springboot 4.X由于其底层依赖和API有较大的变化，所以LiteFlow单独提供了`liteflow-spring-boot4-starter`这个依赖来进行适配，请勿在Springboot 4.X中使用`liteflow-spring-boot-starter`。
:::

如果你想使用Springboot快速开始学习，请参考[Springboot场景安装运行](/pages/9bf6be/)。
