LiteFlow对XML增加了DTD，方便在XML里作一些检查和提醒约束。

具体引用方法为：

```xml {2}
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE flow PUBLIC  "liteflow" "https://liteflow.cc/liteflow.dtd">
<flow>
    <chain name="chain1">
        THEN(a,b,WHEN(c,d));
    </chain>
</flow>
```

:::tip
当然不加也是没有关系的，因为LiteFlow xml节点非常简单，并没有很多的节点标签需要记忆。
:::
